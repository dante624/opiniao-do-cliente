# 📊 Pesquisa de Opinião - TudoWeb
<br><br>
📌 Descrição
<br><br>
Este projeto consiste em um programa desenvolvido em Python para realizar uma pesquisa de opinião sobre o atendimento ao cliente da empresa "TudoWeb".
<br><br>
O sistema utiliza estruturas de repetição (FOR) e condicionais (IF e ELIF) para coletar e analisar respostas de entrevistados.
<br><br>
# ⚙️ Funcionalidades:
<br><br>
🔁 Coleta de dados de 50 entrevistados
<br>
👤 Registro de nome e idade
<br>
⭐ Avaliação do atendimento:
1 = EXCELENTE
<br>
2 = BOM
<br>
3 = RUIM
<br>
📊 Contabilização das respostas:
EXCELENTE ou RUIM
<br>
📢 Exibição dos resultados ao final
<br><br>
